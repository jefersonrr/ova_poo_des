SUBTEMA 2.2 Diseño del Problema

Diseño del Diagrama de Clases sobre una herramienta (UML)

A continuación se listan 3 herramientas para modelado UML

  • StarUML: es una herramienta de código abierto para el modelado de sistemas de software utilizando el lenguaje de modelado unificado (UML). Ofrece una amplia gama de características para el diseño de diagramas de clases, diagramas de secuencia, diagramas de actividad, etc.

  • Visual Paradigm: es una herramienta comercial para el modelado y diseño de sistemas de software utilizando UML. Cuenta con una interfaz fácil de usar y una amplia variedad de funcionalidades para el diseño y documentación de sistemas de software.

  • Umbrello UML Modeller: es una herramienta de código abierto para el modelado de sistemas de software utilizando UML. Ofrece una interfaz intuitiva y una amplia variedad de características para el diseño y documentación de sistemas de software.
