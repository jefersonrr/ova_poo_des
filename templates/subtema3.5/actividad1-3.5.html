Verdadero o Falso de Funciones

 
  1. Una función de asignación es una función que se utiliza para asignar un valor a una variable o a un elemento de una estructura de datos, como una lista o un diccionario. Las funciones de asignación suelen tener la forma "variable = valor" o "estructura_de_datos[índice] = valor".
    1.   verdadero
    2.   falso
