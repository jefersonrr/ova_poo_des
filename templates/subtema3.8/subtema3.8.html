SUBTEMA 3.8 La referencia This

La palabra clave this puede ser usada para invocar a un constructor. Sin embargo, su uso quizás más frecuente en Java tiene lugar en otro contexto: cuando existe sobrecarga de nombres. La sobrecarga de nombres se da cuando tenemos una variable local de un método o constructor, o un parámetro formal de un método o constructor, con un nombre idéntico al de un campo de la clase.

Revisar la bibliografía de Java para mator información: Using the this Keyword (The Java™ Tutorials > Learning the Java Language > Classes and Objects)
