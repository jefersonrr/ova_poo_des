SUBTEMA 3.4 Instrucción de retorno

La instrucción return retorna un valor de una función o método, para que una función o método retorne un valor la definición de cada una de ellas se realiza de diferentes formas. Ya no se usa la instrucción void, sino el tipo de dato que va retornar la función o método, como se muestra en el siguiente ejemplo.

int x; 
                                        //Definición de una función
                                        static int sumar(int a, int b){
                                            return a + b;
                                        }

Al momento de llamar la función tenemos que recuperar el valor que va retornar y eso podemos recuperar en una variable como se muestra en el siguiente ejemplo.

int x; 
                                            public class Main {
                                                public static void main(String[] args) {
                                                    //Llamar a una función
                                                    int r = sumar(10, 20);
                                                    System.out.println("La Suma es: "+ r);
                                                }
                                                //Definición de una función
                                                static int sumar(int a, int b){
                                                    return a + b;
                                                }
                                            }
                                                    //Salida es:
                                                     //La Suma es: 30
